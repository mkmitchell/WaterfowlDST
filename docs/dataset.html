Module Dataset

Defines Dataset class which is initialized by supplying habitat and the crosswalk table. It’s used for organizing spatial datasets.

class waterfowlmodel.dataset.Dataset(inData, scratch, crosswalk=None, classAttr=None)

Dataset class

This creates a model object to store parameters for organizational purposes.

Parameters

  • inData (str) – Dataset input

  • scratch (str) – Scratch geodatabase location

  • crosswalk (str) – CSV file relating input habitat types to ABDU habitat types
